# Auth background videos

Place these files here (Vite serves `public/` at `/`):

| File | Aspect | Used when |
|------|--------|-----------|
| `auth-desktop-16x9.mp4` | 16:9 | Desktop / landscape |
| `auth-mobile-9x16.mp4` | 9:16 | Mobile / portrait (≤900px or portrait tablet) |

Tips: H.264 + AAC (or no audio), keep under ~8MB each, `muted` loop clips work best on iOS.
